# Mapbox (mapbox)

Mapbox is a leading mapping and location data platform that provides tools and services to help developers and businesses create custom maps, visualize geospatial data, and build location-aware applications.

## APIs

### Mapbox Tiling Service
Mapbox Tiling Service (MTS) is a tool for creating vector tilesets, transforming geospatial data into Mapbox-hosted vector tiles.

- **Human URL:** https://docs.mapbox.com/api/maps/mapbox-tiling-service/
- **Base URL:** https://api.mapbox.com

### Mapbox Vector Tiles API
Serves vector tiles from Mapbox-hosted vector tilesets.

- **Human URL:** https://docs.mapbox.com/api/maps/vector-tiles/
- **Base URL:** https://api.mapbox.com

### Mapbox Raster Tiles API
Serves raster tiles generated from satellite imagery and uploaded raster data.

- **Human URL:** https://docs.mapbox.com/api/maps/raster-tiles/
- **Base URL:** https://api.mapbox.com

### Mapbox Static Images API
Serves standalone static map images generated from Mapbox Studio styles.

- **Human URL:** https://docs.mapbox.com/api/maps/static-images/
- **Base URL:** https://api.mapbox.com

### Mapbox Static Tiles API
Serves raster tiles generated from Mapbox Studio styles for use in traditional web mapping libraries.

- **Human URL:** https://docs.mapbox.com/api/maps/static-tiles/
- **Base URL:** https://api.mapbox.com

### Mapbox Styles API
Read and change map styles, fonts, and images. Basis for Mapbox Studio.

- **Human URL:** https://docs.mapbox.com/api/maps/styles/
- **Base URL:** https://api.mapbox.com

### Mapbox Tilequery API
Retrieve data about specific features from a vector tileset based on a given latitude and longitude.

- **Human URL:** https://docs.mapbox.com/api/maps/tilequery/
- **Base URL:** https://api.mapbox.com

### Mapbox Uploads API
Transforms geographic data into tilesets, normalizing projections and generating multi-zoom tiles.

- **Human URL:** https://docs.mapbox.com/api/maps/uploads/
- **Base URL:** https://api.mapbox.com

### Mapbox Datasets API
Read, create, update, and remove GeoJSON features from datasets.

- **Human URL:** https://docs.mapbox.com/api/maps/datasets/
- **Base URL:** https://api.mapbox.com

### Mapbox Fonts API
Accept TrueType and OpenType fonts as raw binary data and generate encoded letters for map renderers.

- **Human URL:** https://docs.mapbox.com/api/maps/fonts/
- **Base URL:** https://api.mapbox.com
